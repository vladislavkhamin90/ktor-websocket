О проект

HomeChat Server - это серверная часть приложения для обмена сообщениями в реальном времени, построенная на Ktor и Kotlin. Сервер предоставляет WebSocket API для подключения Android клиента и обеспечивает рассылку сообщений между всеми подключенными пользователями.

Основные функции

    WebSocket сервер для обмена сообщениями в реальном времени

    Управление подключениями пользователей

    Широковещательная рассылка сообщений всем участникам чата

    Автоматическая очистка отключенных пользователей

    Безопасное соединение с поддержкой SSL/TLS

    Сериализация/десериализация JSON сообщений

Управление соединениями

    Регистрация новых пользователей при подключении

    Удаление пользователей при отключении

    Хранение активных соединений в потокобезопасной мапе

    Логирование подключений/отключений

Технические особенности

    Поддержка WebSocket через Ktor

    Сериализация Kotlinx для работы с JSON

    Потокобезопасные структуры данных (ConcurrentHashMap)

    Конфигурация через HOCON/YAML

    Встроенный SSL сервер с самоподписанным сертификатом
    
  Технологический стек

    Kotlin - основной язык разработки

    Ktor - асинхронный веб-фреймворк

    Kotlinx Serialization - сериализация JSON

    Netty - высокопроизводительный сетевой сервер

    HOCON/YAML - конфигурационные файлы
